Vorläufiges Handbuch zu LasEs

[Johann Schicho]

Login Prozess

Login

Auf der Loginseite können Sie sich mit Ihrer E-Mailadresse und Passwort anmelden. Nach Eingabe der Daten gelangen Sie durch Klicken des Knopfes "Login" auf die Homepage.
Sollten Sie noch kein Benutzerkonto besitzen, so müssen Sie sich zuvor registrieren.

Registrieren

Hier können Sie sich bei dem System erstmalig registrieren.
Dafür müssen Sie Ihre E-Mailadresse, vollständigen Namen und ein neues Passwort angeben. Durch Klicken des Knopfes "Registrieren" werden Sie registriert.
Anschließend erhalten Sie eine E-Mail an Ihre zuvor genannte E-Mailadresse, um sicher zu gehen, dass Ihre E-Mailadresse valide ist.
In der versandten E-Mail befindet sich ein Bestätigungslink. Durch Aufrufen des Links wird ihr Benutzerkonto verifiziert und final erstellt und Sie können sich fortan mit ihrer E-Mailadresse und Passwort anmelden.

Verifizierung

Auf diese Seite gelangen Sie durch den Bestätigungslink in der Ihnen zugesandten E-Mail. Sobald Sie diese Seite aufrufen, ist Ihr Konto verifiziert. Ihr Konto ist nun vollständig einsetzbar. Anschließend werden Sie automatisch auf die Homepage weitergeleitet.

Kopf- und Fußzeile

Navigation und Kopfzeile

Die Kopfzeile ist nur als angemeldeter Nutzer sichtbar am oberen Bildschirmrand. Über die Kopfzeile können Sie durch die Anwendung navigieren. Durch Klicken des Logos gelangen Sie immer zurück zur Homepage. Außerdem können Sie Ihr Profil aufrufen. Über die Suchleiste können Sie Journale und Konferenzen, sowie Ihre Einreichungen finden. Darüber hinaus können Sie auch eine Liste aller Journale und Konferenzen aufrufen. Im Folgenden wird "Journale und Konferenzen" als (wissenschaftliches) Forum abgekürzt.

Fußzeile

Die Fußzeile befindet sich immer am unteren Bildschirmrand. Hierüber erreichen Sie das Impressum.

Impressum

Im Impressum finden Sie das Impressum der Firma, die LasEs einsetzt.

Funktionen und Übersicht

Die einzelnen Nutzergruppen sind nach ihren Nutzungsrechten und Bearbeitungsberechtigungen auf der Website kaskadierend aufgebaut. Das heißt die Funktionen, die einem einfachen Benutzer zur Verfügung stehen, stehen auch einem Gutachter, Editor und Administrator zur Verfügung.

Dabei ist die Hierarchie so aufgebaut: Nutzer < Gutachter < Editor < Administrator.

Wobei eben Links die Benutzergruppe mit weniger Rechten und Funktion steht und Rechts die Gruppe mit allen Rechten und Funktionen.
In diesem Handbuch werden also Funktionen, die der einfachen Nutzergruppe zur Verfügung stehen, nicht nochmals explizit bei höherrangigen Gruppen erwähnt, sondern sind implizit inkludiert.

Funktionen und Buttons werden durch Tooltips in der Anwendung nochmal individuell erklärt. Damit werden einzelne Fragen, die ein Nutzer haben könnte, direkt am richtigen Ort beantwortet.

Funktionen Nutzer

Listen

Im Folgenden wird häufiger über Listen gesprochen. Alle Listen in LasEs sind nach ihren einzelnen Spalten auf- oder absteigend sortierbar. Außerdem sind die Listen nach ihren Kategorien filterbar. Beispielsweise kann eine Liste mit allen Foren nach einem bestimmten Thema, z.B. Informatik, gefiltert werden. Darüber hinaus sind lange Listen paginiert. Nach je 25 Listeneinträgen werden die nächsten Listeneinträge durch Klicken auf die nächste Seite angezeigt.

Homepage

Hier gibt es eine Übersicht über die eigenen Einreichungen in einer Liste.
Durch Klicken auf eine Einreichung kann die Übersicht über den aktuellen Einreichungsprozess aufgerufen werden.

Neue Einreichung

Hier kann ein Paper eingereicht werden. Hier können Sie Ihr Paper als PDF Datei hochladen, den Titel des Papers, E-Mailadressen der Ko-Autoren angeben und einen Editor festlegen. Alle Felder müssen ausgefüllt sein, bis auf die optionalen Ko-Autoren.
Durch Drücken des Submit-Knopfes wird das Paper dann eingereicht und in Zukunft begutachtet.

Einreichungsübersicht

Hier ist die Einreichungsübersicht über ein eigenes Paper. In der Liste werden alle zugehörigen Einträge zum Annahmeverfahren gelistet. Der erste Eintrag ist Ihr eigenes Paper. Eine Art Ampelsystem zeigt den Fortschritt Ihres Papers im Einreichungsverfahren.
Der Indikatorpunkt ist farblos direkt nach der Einreichung. Nach Eingang der Gutachten ist die Ampel gelb, falls Änderungen verlangt werden. Die Ampel ist grün, wenn die Einreichung akzeptiert ist. Die Ampel ist rot, wenn die Einreichung abgelehnt worden ist. Alle folgenden Einträge in der Liste sind fertiggestellte Gutachten von Peers.

Profil

Über die Kopfzeile oben rechts gelangen sie auf Ihr Profil. Hier können Sie Ihren hinterlegten Namen, E-Mailadresse und Passwort ändern.

Suche

In der Kopfzeile gibt es eine Suchfunktion. Darüber können sie wissenschaftliche Foren und Ihre eigenen Einreichungen finden. Alle Ergebnisse werden in einer Liste angezeigt.

Liste der Journale und Konferenzen

Hier finden Sie eine Liste aller wissenschaftlichen Foren. Durch Anklicken können Sie zu den einzelnen Foren navigieren.

Ansicht eines Journals oder Konferenz

Hier können Sie die Kurzbeschreibung lesen, die Website des Forums finden und erhalten eine Übersicht über alle Papers, die Sie zu diesem Forum eingereicht haben. Außerdem können Sie von hier aus ein Paper zu diesem wissenschaftlichen Forum einreichen.

Funktionen Gutachter

Homepage Gutachter

Zusätzlich zur Übersicht über Ihre eigenen Einreichungen können Sie über den Reiter über der Liste auf die Liste der Einreichungen umschalten, für welche Sie Gutachten schreiben sollen. Dabei sehen Sie den Status der Einreichung und gelangen durch Klicken auf die jeweilige Einreichungsseite.

Gutachten einreichen

Auf der Übersichtsseite einer Einreichung können Sie als Gutachter eine Begutachtung einreichen. Auf der Gutachteneinreichungsseite können Sie eine PDF-Datei hochladen und einen kurzen Kommentar verfassen und anschließend das Gutachten mit einer Ja/Nein-Empfehlung für den Editor abspeichern.

Funktionen Editor

Homepage Editor

Zusätzlich gelangen Sie über den dritten Reiter über der Liste zu der Übersicht aller Einreichungen, denen Sie als Editor zugeordnet sind. Hier können Sie dann zu den einzelnen Einreichungen navigieren.

Ansicht eines Journals oder Konferenz

Als Editor eines Ihrer Foren können Sie weitere Nutzer zu Editoren ernennen. Ebenso können Sie die Kurzbeschreibung und den Link zur Website ändern.

Nutzerliste

Hier können Sie die einzelnen Nutzer und deren vollständigen Namen und E-Mailadresse einsehen.
Damit können Sie die E-Mailadresse von möglichen Gutachtern finden und ihr Profil einsehen.

Einreichungsübersicht Editor

Bei der Übersicht einer Einreichung, welcher Sie als Editor zugeteilt sind, übernehmen Sie die Verwaltung. Sie können den verwaltenden Editor ändern, Gutachter per E-Mailadresse berufen und eingereichte Gutachten zur Einsicht für den Einreicher freischalten. Außerdem können Sie eine Revision verlangen. Abschließend können Sie die Einreichung annehmen oder ablehnen.

Funktionen Administrator

Administration

Hier werden die Einstellungen der Website geändert. Man kann das Impressum bearbeiten, das Farbschema ändern und neue Foren anlegen.

Neues Journal/Konferenz

Um ein neues Forum anzulegen müssen Sie mehrere Informationen angeben. Es muss mindestens einen initialen Editor geben. Außerdem können Sie eine Kurzbeschreibung hinzufügen und die zugehörige Website des wissenschaftlichen Forums verlinken.

Nutzerliste

Hier können Sie die einzelnen Nutzer und deren vollständigen Namen und E-Mailadresse einsehen.
Außerdem können Sie hier Nutzer löschen oder neue Nutzer hinzufügen.

Nutzer hinzufügen

Dafür braucht es im Gegensatz zum normalen Registrierungsprozess keine E-Mail Bestätigung. Es müssen ebenfalls eine E-Mailadresse, der vollständige Namen und ein Passwort angegeben werden. Dieser Account kann dann an Dritte zur Benutzung weitergegeben werden.
